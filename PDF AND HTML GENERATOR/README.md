# Gerador Automático de Relatórios (HTML e PDF)

Projeto em Python que lê dados de um arquivo CSV, realiza análise estatística,
gera gráficos e exporta relatórios automaticamente em HTML e PDF.

## Funcionalidades
- Leitura de dados CSV
- Análise automática com Pandas
- Geração de gráficos com Matplotlib
- Relatório em HTML
- Relatório em PDF (ReportLab)

## Tecnologias
- Python
- Pandas
- Matplotlib
- ReportLab
